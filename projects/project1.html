Data Fest Analysis

This project analyzed dataset X using Python and machine learning techniques to predict Y. The goal was to extract actionable insights from raw data and visualize trends effectively.

Key Highlights

  • Awarded Best Visualization at DataFest 2024 for innovative visual analytics of student learning data
  • Analyzed anonymized log‑data from CourseKata (11 universities, 48 classes, >1,600 students) to explore how success across question types influences overall student performance
  • Created visualizations linking question-type performance to student outcomes, identifying patterns to improve engagement and learning strategies
